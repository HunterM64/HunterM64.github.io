Command line password manager written in Rust. SQLite is used to keep track of user data, which includes their master password made after running PassMan for the first time, and all saved passwords and the websites associated with them. The master password is hashed using Argon2 and the website passwords are encrypted with AES-256. Available subcommands for PassMan include generate, list, update, and delete, providing basic CRUD functionality.
